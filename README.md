# crop_image_chrome_extension
国内让AI画个图老是底下有 AI生成 水印，烦的一比，这个chrome插件就是为了裁剪掉那块的水印而生。
## 使用方法
下载zip压缩包之后解压出来，然后打开chrome扩展页，点击加载已解压的扩展，然后在弹出的窗口中选中刚刚解压的根文件夹
